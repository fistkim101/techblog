# 만들면서 배우는 클린 아키텍처

클린 아키텍처 읽기 전에 먼저 진행했다. 하루 날잡고 읽었는데 딱 하루(오전 8시 30분쯤 부터 시작해서 저녁 8시 정도에 끝)만에 다 봐질만큼 책이 그렇게 두껍지도 않고 내용이 복잡하지도 않다. 다만 제대로 알기 위해서는 책 제목처럼 직접 '만들면서' 익혀야 이해도가 높아지는 것 같다.

이번 정리는 책을 다 읽고 예제 구현까지 완료한 상태에서 정리를 시작한다.
